ランテル語 文法解説

概要 音素・音韻 形態論

音韻体系の基本性格

ランテル語は 制約主導型・軽量音節志向の膠着語的音韻体系 を持つ。

設計思想：

  • 流動性（母音中心）
  • 輪郭保持（語頭制約）
  • 子音による境界明示
  • 呼気による時間標識

音素体系

母音

前舌中央後舌備考
iə（弱勢）u強勢核は e / i / u に置かれやすい
e長母音は独立音素というより融合で生じる

子音（概略）

カテゴリ音素メモ
破裂音p, b, t, d, k, g
摩擦音s, z, h
鼻音m, n語幹末 -n の処理は特別
流音r, l
特殊c, q口蓋垂摩擦音

備考：半母音的機能は母音融合で表れる（書記上は独立しない）。語頭単独無声破裂音は制限される。

音節構造と語頭制約

基本音節型： (C)V(C)

  • CCCは存在しない / CCは最大2子音
  • 語頭VCCは同一無声子音のみ許可
  • 語頭VCの場合、Cは有声子音のみ

語頭VC制限

語頭が VC の場合、C は必ず有声。

例：ignes / ort / edor

語頭VCC制限

語頭が VCC の場合、CC が同一無声子音であれば許可。

例：accer, app…（想定形）

中央母音 /ə/ と母音連続

/ə/ の性質

  • 多音節語の中間に出現（語末は原則なし）
  • 強勢を担わない（緩衝母音）

母音連続の扱い

母音連続は許容。ただし既存の二重母音規則に一致する場合のみ音変化が起こる。

二重母音（綴り）
ai / au / ei / eu / eo
iu / ua / ui / ue / uo / ou

例：ke + i → kei → /kiː/、mata + i → matai → /mate/

重要：語幹は変化せず、音韻処理のみ。

特殊規則（鼻音保持・過去子音弱化）

鼻音保持規則

語幹が -n で終わる場合、Xn + V → XnnV

例：an + e → anne、fon + i → fonni

過去を示す -c の弱化

-c は /χ/ → /h/ → 呼気化へ弱化しやすいが、完全脱落は避けられる（時制区別のため）。
